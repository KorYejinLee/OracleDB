# 파이썬 웹크롤링 및 Oracle DB 데이터 삽입
## 개요
### 1. Gmarket 웹사이트에서 iphone 관련 데이터를 스크래핑하여 CSV 파일로 저장하는 Python 스크립트
### 2. CSV 파일을 읽고 Oracle 데이터베이스에 데이터를 삽입하는 Java 코드
